# Frontend

Web UI for the Smart Watch hiking tracker. Displays active and past hiking sessions, allows deleting sessions, and lets users configure their weight. Shows real-time watch connection status.

Built with React, TypeScript, and Vite.

## Features

- Active hiking session card (steps, calories, distance)
- Expandable list of past sessions with details
- Delete past sessions
- Collapsible settings panel for weight configuration
- Watch connection status indicator

## Development

```bash
npm install
npm run dev
```
